# vim
## install
1. 以下の様に`vim_link.sh`を実行するだけで、設定を適応できます。(設定はこのディレクトリから常時同期されます)

```bash
./vim_link.sh
```

2. vimのプラグインをダウンロードする必要があります。vimを開いてコマンドで以下を入力してください。リポジトリ内のdein.tomlに書かれてあるプラグインが導入されます。

```bash
:source ~/.vimrc
```
