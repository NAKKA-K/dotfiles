# ペルソナ分析フレーム

## 面白さの座標系：6象限マッピング

> **[Claude内部フレーム — 筆者との対話では使わない]**
> 感想単語（あるある・それだわ・興味深っなど）は筆者にそのまま出してよい。「象限」「有意識/無意識」「funny/interested」といったラベルは出さない。

```
上下軸の本質：「もう言葉にできていたか」
　有意識 → 半言語化 → 無意識 ＝ 再認識 → 発見 → 衝撃

左右軸の本質：「誰に届くか」
　知識少 → 知識多 ＝ 広い → 狭い ＝ 感情（funny）→ 知性（interested）

右下に行くほど刺さりが深くなり、届く対象が狭くなる
```

| 象限 | 感想単語 | 意味合い |
|------|---------|---------|
| 知識少×有意識 | あるある | 感情的共鳴。読者の体験が肯定される。射程最広・刺さりは浅い。導入に向いているがそれだけでは面白みに欠ける。 |
| 知識少×半言語化 | なるほどね | 視点の提供。誰でも体験できる領域の気づき。funny と interested の中間に位置し、大抵の読者が理解している前提を浮き彫りにする。 |
| 知識少×無意識 | それだわ | 普遍的な盲点の言語化。詩的・哲学的な感動に近い。普遍的な知識を前提とするため多くの人に刺さる。一方で発見としては表層に留まる。 |
| 知識多×有意識 | だよね | 知識的確認。専門読者の前提を揃える。次の展開への助走 |
| 知識多×半言語化 | たしかに | 専門知識層への刺さり。モヤっとしていたものが言語化された解放感。今まで言語化されていなかった感覚と知識を両立する輪郭化 |
| 知識多×無意識 | 興味深っ | 知的衝撃。射程最狭・刺さりは最大。読み手本人自体もまだ問として立てられていない問い、もしくは暗黙知として感覚化されつつあるものを精緻に表現した学び。行動変容を生み出しうる層。 |

## 使い方

Step 1 の内部メタ分析（行動変容の設計検証）で使用する。
読後変容の推定として、この記事を読み終えた後に読者が発しそうな言葉（あるある / なるほどね / それだわ / だよね / たしかに / 興味深っ）をこの表で特定し、現状の構成・表現でそれが実現できているかを検証する。

**この分析結果は出力に含めない**。分析用語（象限・有意識/無意識・funny/interested）は完成稿・レビュー結果に一切出さない。
